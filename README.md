hn
=========
read hacker news from the command line


### Install

```shell
make install
```

### Usage

1. Start the daemon
2. Use the cli interface

```shell
# Start the daemon
$ hncd

# Use the cli interface
$ hnctl
ctl > top  # show top stories
ctl > new  # show new stories

# Or you can invoke the command directly
$ hnctl top
```
